# SDK Android

### Instalação
Primeiro, adicione o nosso repositório no `build.gradle` do seu projeto:
```groovy
flatDir {
  dirs 'libs'
}
```

depois, adicione como dependência no módulo desejado:
```groovy
dependencies {
  compile(name: 'stone-sdk-2.3.0-debug', ext: 'aar')
}
```
